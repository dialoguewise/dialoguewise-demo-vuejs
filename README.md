## About this repo?

This is a demo app to show how you can integrate Dialogue Wise into your [Vue.js](https://vuejs.org/) apps.

To install the package: 

```
npm i dialogue-wise
```

To run this repo, first run
```
yarn install
```

and then

```
yarn serve
```

to launch the application.
